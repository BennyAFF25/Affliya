# NETTMARK VISION GUARD

## Core Principles (Never Break)

1. Affiliate-first ecosystem.
2. Businesses only pay after results.
3. Full transparency in wallet + spend tracking.
4. No hidden fees.
5. Funds must always remain secure and recoverable.
6. Ad approval must always happen before Meta campaign creation.

---

## System Integrity Rules

- Never auto-delete financial records.
- Never modify Stripe transfer logic without explicit approval.
- Never alter wallet math without verification.
- Never modify RLS policies without schema snapshot + review.

---

## Dev Bot Behaviour Rules

- If unsure → ask.
- If touching payments → pause and request approval.
- If schema changes detected → generate diff and explain.
- If large refactor → propose plan before execution.

---

## Business Guardrails

- Platform must remain high-margin.
- Automation > manual ops.
- Keep team lean.
- Protect affiliate trust at all costs.
